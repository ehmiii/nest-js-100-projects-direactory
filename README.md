## 🚀 **QUICK START GUIDE**

<div align="center">

### ⚡ **GET STARTED IN 60 SECONDS**

</div>

```bash
# 🎯 Step 1: Choose a project and clone it
git clone https://github.com/your-username/nestjs-expense-tracker.git
cd nestjs-expense-tracker

# 🎯 Step 2: Install dependencies  
npm install

# 🎯 Step 3: Setup environment
cp .env.example .env
# Edit your .env file with database credentials

# 🎯 Step 4: Start with Docker (Recommended)
docker-compose up -d

# 🎯 Step 5: Run the application
npm run start:dev

# 🎉 Your API is live at http://localhost:3000
# 📖 Swagger docs at http://localhost:3000/api
```
